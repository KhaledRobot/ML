⚖️ AI Trial Simulation System

A Python-based courtroom trial simulation system using AI-powered agents representing defense, prosecution, witnesses, and a judge. This project automates trial sequences based on structured case descriptions, generating realistic opening statements, interrogations, and verdicts.

 Features

    AI-driven lawyer agents for prosecution, defense, witness, and judge roles.

    Automated trial phases: opening statements, interrogations, closing statements, verdict.

    Batch execution on 600+ case descriptions from a CSV dataset.

    Outputs simulated court transcripts per case.
